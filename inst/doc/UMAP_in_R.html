UMAP in R

2018-05-21

Uniform Manifold Approximation and Projection (UMAP) is a non-linear dimensionality reduction algorithm. It is similar to t-SNE but computationally more efficient. UMAP was created by Leland McInnes and John Healy (github, arxiv). umapr wraps the Python implementation of UMAP to make the algorithm accessible from within R.

Basic use

Here is an example of running UMAP on the iris data set.

library(umapr)
# select only numeric columns
embedding <- as.data.frame(umap(as.matrix(iris[ , 1:4])))

# look at result
head(embedding)
##            V1       V2
## 1  0.15531684 11.25818
## 2 -0.09238198 12.90883
## 3 -0.66098781 12.90071
## 4 -0.45995370 13.28126
## 5 -0.14524970 11.21500
## 6  0.42218236 10.15282
# plot result
library(ggplot2)
ggplot(embedding, aes(x = V1, y = V2, color = iris$Species)) + 
  geom_point()

umap returns a data frame with two columns containing the UMAP embeddings of the original data and (optionally) the original data attached.
